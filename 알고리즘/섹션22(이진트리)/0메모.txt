리스트 : 선형 -> 한가지 선택
트리 : 비선형 -> 여러가지 선택을 할수 있다.
트리의 규칙 : 무조건 자식노드만 가리켜야한다. 부모나 형제를 가리킬수 없음
root : 꼭대기 노드
child : 자식 노드
parent : 부모 노드
siblings : 부모가 같은 자식들
leaf : 자식이 없는 노드
edge : 간선 -> 노드에서 다른 노드로 향하는 화살표

트리의 예시 
HTML의 DOM
네트워크 라우팅
추상 구문 트리
인공지능
운영체제에서 폴더가 설계된 방식
JSON

이진 트리 : 자식이 최대 2개여야 한다. 3개이상이면 이진 트리가 아니다.
왼쪽엔 작은값, 오른쪽엔 큰값을 배치한다.
